## 📌 Sobre o projeto

A busca manual de coordenadas geográficas a partir de um CEP é uma tarefa repetitiva e propensa a erros — especialmente em projetos que lidam com grandes volumes de endereços. Este projeto nasceu para resolver exatamente isso: **automatizar a conversão de CEPs em dados geográficos**, tornando o desenvolvimento de mapas mais ágil e confiável.

Com ele, você consegue integrar localização geográfica ao seu fluxo de trabalho sem precisar fazer consultas manuais, economizando tempo e reduzindo retrabalho.

### 🛠️ Tecnologias utilizadas

| Tecnologia | Finalidade |
|---|---|
| **Python** | Linguagem principal do projeto |
| **Requests** | Consulta à API de CEP via HTTP |
| **Pandas** | Manipulação e organização dos dados |
| **GeoPandas** | Tratamento de dados geoespaciais |

## 🚀 Como utilizar

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
cd seu-repositorio
```

### 2. Crie e ative o ambiente virtual

```bash
# Linux / macOS
python -m venv venv
source venv/bin/activate

# Windows
venv\Scripts\activate
```

> O terminal exibirá `(venv)` no início da linha quando o ambiente estiver ativo.

### 3. Instale as dependências

```bash
pip install -r requirements.txt
```
